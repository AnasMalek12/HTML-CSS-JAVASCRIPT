# 🐱 Cat Facts (API Project)

_A simple web app that fetches random **cat facts and images** using an API._  
Built with **HTML**, **CSS**, and **JavaScript**.

## 🌐 Live Demo  
👉 **[🌐 View the live site](https://anasmalek12.github.io/HTML-CSS-JAVASCRIPT/Cat%20Facts%20(With%20API))**

## 🛠️ Technologies Used
- HTML5  
- CSS3  
- JavaScript (Fetch API)  

## ✨ Features
- 🐾 Get random cat facts  
- 🖼️ Display cat images from API  
- 🔄 Fetch new data on button click  
- ⚡ Fast and simple UI
